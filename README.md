# 🌙 Known Stranger (아는 사이 X)
AI Mystery Romance Game

An interactive narrative game where players reconnect with their ex through anonymous midnight conversations.

## 🎮 Live Demo
https://muffinbiter.github.io/known-x/

## 📖 Overview
Known Stranger is a mystery romance game powered by AI character simulation.

Players describe their ex-partner through a short survey.  
The system recreates their personality and conversation style.

Three anonymous strangers then appear in a late-night chat app.

Through conversation clues, players must identify **which one is their ex.**

The game combines **AI character simulation, narrative gameplay, and mystery solving.**

## ✨ Core Features
- AI-generated ex-partner personality simulation  
- Anonymous chat mystery gameplay  
- Relationship affection system  
- Conversation-based clue collection  
- Action point system for daily interactions  
- Mobile-first chat UI

## 🛠 Tech Stack
- HTML5  
- CSS3
